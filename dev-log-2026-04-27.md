# Diary App — Dev Log, April 27 2026

## What we built today

Started the diary app from scratch. Here's everything that happened:

### Setup
- Installed GitHub CLI and connected your GitHub account (artemostapchuk343)
- Created the `diary-app` repository on GitHub
- Installed Node.js and scaffolded the project with Vite + React

### Core app
- **Lock screen** — password protection using AES-256-GCM encryption via the browser's Web Crypto API. No server involved, your password never leaves your device.
- **Entry list** — shows all entries newest first, with live search by title or body text
- **Entry editor** — write entries with a title, body, mood emoji, and file/photo attachments
- **Local storage** — everything stored in IndexedDB via Dexie, works fully offline

### UI
- Dark theme throughout
- Mood picker — a button that opens a popover with 60 emojis
- Attachment support — photos show as thumbnails, other files show as name + icon

### Google Drive sync
- Connected Google OAuth (GIS library) using your Google Cloud project
- Entries sync to a `My Diary` folder in your Google Drive as `.md` (Markdown) files
- Each file has frontmatter (date, title, mood, timestamps) — readable by any AI
- Sync is manual — press the Sync button whenever you want to push/pull
